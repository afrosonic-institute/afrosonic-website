## [0.1.0] — 2026-05-08

Initial commit. Captures the v0.app / Vercel scaffold and the first hand-written static pages.

### Added
- **Project scaffold** generated by [v0.app](https://v0.app):
  - Next.js 16 (App Router) with React 19
  - TypeScript 5.7 in strict mode (`noEmit: true`, `ignoreBuildErrors: true`)
  - Tailwind CSS 4 via `@tailwindcss/postcss`
  - shadcn/ui (`new-york` style, neutral base colour) — 60+ Radix-based components in `components/ui/`
  - `lucide-react` icon library
  - `@vercel/analytics` (production-only, mounted in `app/layout.tsx`)
  - pnpm as package manager (`pnpm-lock.yaml` committed)
- **Routing layer** (`app/`):
  - `app/layout.tsx` — root layout with Geist + Geist Mono fonts, favicons (light/dark/SVG/Apple), Vercel Analytics
  - `app/page.tsx` — redirects `/` → `/index.html`
  - `app/globals.css` — Tailwind base + shadcn design tokens (oklch colour space)
- **Static site** in `public/`:
  - Seven pages: `index.html`, `rawa.html`, `film-club.html`, `editorial.html`, `archive.html`, `about.html`, `support.html`
  - Hand-written `public/styles.css` implementing the Afrosonic design system
  - Google Fonts: IBM Plex Serif (display), Source Serif 4 (body), IBM Plex Sans (UI)
  - Hamburger nav with `aria-expanded` and `aria-label` for mobile
  - Hero, "What We Do", credibility bar, programme cards, featured content sections wired up on the homepage
  - Placeholder ticket CTAs and event metadata
- **Brand assets**:
  - `public/icon.svg`, `public/icon-light-32x32.png`, `public/icon-dark-32x32.png`, `public/apple-icon.png`
  - Placeholder images: `placeholder.svg`, `placeholder.jpg`, `placeholder-logo.png`, `placeholder-logo.svg`, `placeholder-user.jpg`
- **Configuration**:
  - `next.config.mjs` — `typescript.ignoreBuildErrors: true`, `images.unoptimized: true`
  - `components.json` — shadcn config with `@/*` path aliases
  - `tsconfig.json` — strict mode, `module: esnext`, `jsx: react-jsx`, paths alias `@/* → ./*`
  - `postcss.config.mjs` — Tailwind 4 plugin
- **Documentation** (this commit):
  - `README.md` — project overview, architecture, local dev, deploy, file structure
  - `CHANGELOG.md` — this file
  - `CLAUDE.md` — project context, design system, decisions log for Claude sessions
  - `.gitignore` — Next.js + pnpm + editor + OS junk

### Architecture note

The repo is intentionally a **Next.js wrapper around a static HTML site**, not a typical Next.js app. All content lives in `/public/`. `app/page.tsx` exists only to redirect the root URL to `/index.html`. The shadcn/ui components and Tailwind utilities are bundled but unused on the live site.

### Known content placeholders (carried forward)

- Paystack donation link
